The Cascade Algorithm

  • Specificity: Inline style(1000) > ID selector(100) > Class or Attribute or Pseudo Selector(10) > Element Selector(1) > Universal Selector(0)
  • Cascade: for the selector with the same specificity the selector declared later has priority.
  • !important keyword: this has the highest specificity among the all (consider the specificity valuue as 10,000).
